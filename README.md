# TaskManager - PHP Laravel Task Management System

A production-ready Laravel application featuring a Task Management system with Tailwind CSS styling, automated database seeding, and deployment readiness for platforms like Railway.

## Features
-   **Task Tracking**: Title, Description, Status (Pending/Completed), Priority (Low/Medium/High), and Due Dates.
-   **Automated Seeding**: Generates 20+ realistic tasks and 3 demo users automatically.
-   **Tailwind CSS UI**: Modern, responsive layout with striped tables, colored badges, and indigo-themed buttons.
-   **Deployment Ready**: includes `Procfile` for Railway/Heroku and SQLite support.

## Prerequisites
-   PHP >= 8.2
-   Composer
-   SQLite (Standard on macOS/Linux and most Windows PHP stacks)
